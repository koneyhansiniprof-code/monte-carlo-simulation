# monte-carlo-simulation-prices
This program is a financial engineering tool designed to price an Up-and-Out Barrier Call Option using a Monte Carlo simulation. It incorporates advanced market features like Stochastic Volatility (switching between high and low volatility states) and Jump Diffusion (sudden spikes or drops in price).The code allows users to either simulate these prices based on mathematical models or input prices manually. It uses a Selection Sort algorithm to organize the data and provides analysis tools like histograms and price summaries.
