# monte-carlo-simulation-prices
This program is a financial engineering tool designed to price an Up-and-Out Barrier Call Option using a Monte Carlo simulation. It incorporates advanced market features like Stochastic Volatility (switching between high and low volatility states) and Jump Diffusion (sudden spikes or drops in price).The code allows users to either simulate these prices based on mathematical models or input prices manually. It uses a Selection Sort algorithm to organize the data and provides analysis tools like histograms and price summaries.Key Components1. Class Structure: PriceProcessorThe entire logic is encapsulated within the PriceProcessor class, which manages:Parameters: Financial inputs like Spot Price ($S_0$), Strike Price ($K$), Barrier Level, Interest Rate ($r$), and Time to Maturity ($T$).Data Storage: A dynamically allocated array (price_data) to store the generated or inputted simulation prices.State Management: A boolean (data_ready) to track if data exists before attempting to sort or analyze it.2. Simulation Logic: run_monte_carlo()This is the core financial engine.Detailed Output: It provides a step-by-step log for the first few paths, showing the stock price movement, volatility changes, jump events, and barrier breach events.Antithetic Variates: Although not explicitly utilizing a second path in the for loop, the structure is set up to handle large datasets.Knock-Out Logic: If the price hits the barrier level, the final price for that simulation path is set to $0.0$.Path Logging: For the first three paths, it prints specific steps where jumps occurred or where the option was knocked out.3. Sorting Algorithm: selection_sort()Algorithm: Implements a classic Selection Sort, which has a time complexity of $O(n^2)$.Usage: It is called automatically before histogram generation, summary calculations, or saving to a file to ensure data is ordered from lowest to highest.4. Menu System: main()Provides a command-line interface with the following options:Manual Input: User enters a custom number of prices.Monte Carlo: Runs the simulation with detailed log output.Show Histogram: Visualizes the distribution of terminal prices using ASCII characters.Price Summary: Calculates Min, Max, and Average prices.Save to File: Exports the sorted prices to sorted_prices.txt.Find Better Price: Identifies the best price based on whether the user is buying (lowest price) or selling (highest price).
